# ------------------------------- # Arithmetic Functions # ------------------------------- def add(a, b): return a + b def subtract(a, b): return a - b def multiply(a, b): return a * b def divide(a, b): if b == 0: raise ValueError("Cannot divide by zero.") return a / b # ------------------------------- # Input Handling # ------------------------------- def get_number(prompt): while True: try: return float(input(prompt)) except ValueError: print("⚠️ Invalid number. Try again.") def get_operation(): operations = ["+", "-", "*", "/"] while True: op = input("Choose operation (+, -, *, /): ") if op in operations: return op print("⚠️ Invalid operation.") # ------------------------------- # Core Logic # ------------------------------- def calculate(a, b, operation): if operation == "+": return add(a, b) elif operation == "-": return subtract(a, b) elif operation == "*": return multiply(a, b) elif operation == "/": return divide(a, b) # ------------------------------- # Main Program # ------------------------------- def main(): print("🧮 Simple Calculator") while True: num1 = get_number("Enter first number: ") operation = get_operation() num2 = get_number("Enter second number: ") try: result = calculate(num1, num2, operation) print(f"✅ Result: {result}") except Exception as e: print(f"❌ Error: {e}") again = input("\nDo another calculation? (y/n): ").lower() if again != "y": print("Goodbye 👋") break if __name__ == "__main__": main()
